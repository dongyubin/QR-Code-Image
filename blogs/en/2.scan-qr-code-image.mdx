## Why Scan a QR Code Image?

People often receive QR codes as screenshots, photos, PDFs, chat images, or downloaded files. Instead of using a phone camera on another screen, a browser-based scanner can read the QR code image directly from an upload.

This is useful when the code appears in a desktop screenshot, a saved menu, a presentation slide, a receipt, a product label, or an image sent through chat.

## Best Way to Scan a QR Code Image

Use this workflow when decoding a QR code image:

1. Save the screenshot or photo.
2. Open a QR code image scanner.
3. Upload the image file.
4. Check the decoded result before opening the link.
5. If the scanner fails, crop closer to the QR code and try again.

A good scanner should handle PNG, JPG, JPEG, and common screenshot formats. For privacy, browser-first scanning is preferable because the image can be decoded locally without uploading it to a server.

## Why a QR Code Image May Not Be Detected

If you see an error like "No readers were able to detect the code", the scanner is usually saying that it cannot find a valid QR structure in the image. The root cause is often visual, not the text payload.

Common causes include:

- The screenshot is blurry or compressed.
- The QR code is cropped and missing the quiet zone.
- The QR code is too small in the image.
- The foreground and background colors have weak contrast.
- The position markers are overly decorative.
- A styled module shape removes too much dark signal.
- The image has glare, shadows, rotation, or perspective distortion.

## How to Fix a QR Code Image That Will Not Scan

Try these fixes in order:

### Crop the image

Remove extra background and keep the full QR code visible. Do not cut into the white border around the code.

### Increase the size

If the QR code is tiny inside a screenshot, zoom in or export it again at a larger size.

### Use stronger contrast

Dark modules on a light background are easier to detect. Avoid pale yellow, low-opacity colors, and heavy gradients for the encoded modules.

### Avoid compression

JPG compression can damage small modules. If possible, export the QR code image as PNG or SVG.

### Test the final file

Do not only test the live preview. Test the exact PNG, JPG, or SVG-derived file that you plan to send, print, or publish.

## How to Create QR Code Images That Scan Reliably

If you are creating the QR code image yourself, read the [QR code image creation guide](/blog/qr-code-image) and start with these settings:

- Error correction: Q or H.
- Quiet zone: at least 2 modules, preferably more for print.
- Module density: avoid overly thin modules.
- Position markers: keep them clear and high contrast.
- Background: plain and light.
- Export: SVG for print, PNG for general digital use.

Decorative styles can be useful, but they should not replace the scanning structure. If a style uses dots, lines, cross shapes, or unusual markers, always test it before publishing.

## When to Use Camera Scan Instead

Upload scanning is best for screenshots and saved images. Camera scanning is better when the QR code is printed, displayed on another device, or physically placed on a sign, package, table, badge, or flyer.

The best QR workflow offers both options: scan from camera when the code is in the real world, and scan from image when the code already exists as a file.

## FAQ

### Can a browser scan a QR code image locally?

Yes. Modern browser QR scanners can decode an uploaded image locally when the scanner library runs in the browser.

### Why does my phone scan a QR code but the upload scanner fails?

Different scanners use different detection engines. Some phone cameras apply sharpening, perspective correction, and exposure handling that an upload scanner may not apply.

### Should I use PNG or JPG for a scannable QR code image?

PNG is usually safer. JPG can work, but compression artifacts may reduce readability.

### Can a QR code image be too stylish?

Yes. If decoration weakens the finder markers, timing pattern, quiet zone, or module contrast, scanners may fail even if the code looks visually clear to humans.
